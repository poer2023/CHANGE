# 🎯 模块化编辑器新功能使用指南

## 🔍 如何访问新功能

### 第一步：进入模块化编辑器
1. 启动应用: `npm run dev`
2. 打开浏览器访问: http://localhost:3000
3. 点击主页的 "创建新论文" 或 "编辑器页面"
4. 选择 "模块化编辑器" 模式

### 第二步：体验新功能

#### 📊 依赖关系可视化
- **位置**: 顶部工具栏右侧的网络图标 (🔗)
- **功能**: 显示模块间的依赖关系图表
- **体验**: 点击图标切换到依赖关系视图

#### 🎯 智能模板系统  
- **位置**: 右侧面板 → "模板" 标签页
- **功能**: 
  - 浏览预设学术论文模板
  - 🪄 **智能填充**: 点击紫色魔法棒图标
  - 👁️ **预览**: 点击眼睛图标查看模板结构
- **体验**: 
  1. 选择一个模板
  2. 点击智能填充按钮
  3. 填写研究信息后自动生成内容

#### ✨ 增强模块卡片交互
- **位置**: 中间编辑区域的任意模块
- **新功能**:
  - 🎨 **富文本工具栏**: 编辑时显示格式化选项
  - 📊 **实时统计**: 编辑器右上角的图表图标
  - ⌨️ **快捷键**: Ctrl+B、Ctrl+I、Tab缩进
  - 📈 **进度追踪**: 动态显示字数和完成度

#### 🔄 拖拽重排
- **位置**: 悬停在模块卡片左侧
- **功能**: 出现拖拽手柄，可重新排序
- **体验**: 拖拽模块改变顺序

## 🎯 核心亮点功能

### 1. 智能填充对话框
```
位置: 模板页面 → 点击智能填充按钮
功能: 输入研究信息，AI自动生成论文框架
```

### 2. 依赖关系可视化
```
位置: 顶部工具栏 → 网络图标
功能: 显示模块间的逻辑依赖关系
```

### 3. 富文本编辑器
```
位置: 模块编辑状态
功能: 
- 格式化工具栏
- 实时字数统计
- Markdown支持
- 键盘快捷键
```

### 4. 进度追踪可视化
```
位置: 模块属性面板
功能: 
- 字数进度条
- 完成度百分比
- 阅读时间估算
```
